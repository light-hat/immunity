## Результаты работы

В рамках дипломного проекта:
- разработан агент интерактивного анализа;
- разработан управляющий сервер;
- собран обучающий набор данных, а также выполнено трансферное обучение модели BERT для задачи выявления уязвимостей;
- разработан тестовый стенд, описывающий внедрение системы в процессы DevSecOps;
- проведено сравнительное тестирование разработанной системы с инструментами SAST и DAST.
